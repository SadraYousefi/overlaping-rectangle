### How to use 

1. install dependencies: 

```
npm i
```

2. create .env file

```
DB_URI= your db uri
SYNCHRONIZE_MODE= must be a boolean value

```

### how to run 
1. run by using this command
```
npm run build
npm run start:prod
```
2. run as dev
```
npm run start:dev
```


### api documentaion
1. to access api documentaion use this url : 
```
youraddress:3000/docs
```
